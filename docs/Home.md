# JavaToGpu Wiki

JavaToGpu is a source-first Java-to-OpenCL compiler and runtime stack with an additional structured ASM frontend for advanced integrations.

This wiki is the user-facing documentation set for the project.

Use this wiki if you want to:

- write Java kernels for OpenCL
- understand the supported subset
- use the runtime safely in production-style code
- diagnose compile/runtime failures
- integrate JavaToGpu into a larger toolchain
